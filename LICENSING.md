# Licensing note

This repository currently does not include an open source license from the original upstream project.

That means the code is publicly visible on GitHub, but it should not be treated as permissively licensed until the upstream maintainer adds a license or grants permission.

Recommended next steps:

1. Ask the upstream maintainer to add a license, such as MIT or Apache-2.0, if they intend broad reuse.
2. Avoid adding a new license to a fork unless you have the right to license the original code.
3. If you contribute changes upstream, keep attribution and repository history intact.
